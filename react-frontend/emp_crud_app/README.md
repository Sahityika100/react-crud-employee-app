# Employee CRUD App

A simple full-stack Employee Management system built with **Spring Boot**, **React**, and **MySQL**.

---

## Tech Stack

* Backend: Spring Boot 2.7.15, JPA, MySQL
* Frontend: React, Axios

---

## Setup

### Backend

```bash
cd backend
mvn spring-boot:run
```

Configure your DB in `application.properties`.

---

### Frontend

```bash
cd frontend
npm install
npm start
```

---

## API

Get,Post,Put,Delete
---

## Notes

* `application.properties` is ignored (contains sensitive data)
